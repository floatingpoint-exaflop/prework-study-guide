HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A tag is just the stuff in the operands; the element is those operands and the stuff in between the opener and closer.
  • Tags properly coded don't render on the UI and if they have no contents, the element itself will be invisible too.
  • Meta tags are a versatile and essential method of telling the browser how to handle your webapge.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all files in the repo for the upcoming commit
  • git commit -m "comment": commits branch files to main with title for pull request
  • git pull origin master will pull changes from the origin remote, master branch and merge them to the local checked-out branch.
  • git push sends changes from the specified branch back to main/master.
